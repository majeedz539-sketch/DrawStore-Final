## 🚀 كيفية الاستخدام

### للبدء السريع:
```bash
# 1. تثبيت المكتبات
pip install -r requirements.txt

# 2. تشغيل التطبيق
python app.py

# 3. افتح المتصفح
# http://localhost:5000
```

### اختبار النظام:
1. ستصل إلى صفحة تسجيل الدخول
2. أدخل مفتاح ترخيص صحيح
3. شاهد الملفات المتاحة لك
4. استخدم الفلتر لاختيار نوع الملف
5. احمل الملف
6. اضغط الخروج عندما تنتهي
