# 🎯 DrawStore - دليل النظام الجديد

## ✨ التحديثات الجديدة

### 1️⃣ نظام الترخيص (License Key)
- **قبل الدخول**: يجب إدخال مفتاح الترخيص
- **التحقق**: يتم التحقق من المفتاح عبر Railway API
- **الأمان**: جميع التحميلات محمية بمفتاح صحيح

### 2️⃣ الملفات المخصصة (File Selection)
- **كل مستخدم يرى ملفاته فقط**: بناءً على مفتاح الترخيص
- **فلترة ذكية**: يمكن تصفية الملفات حسب النوع
- **توفير البيانات**: لا يرى المستخدم ملفات لم يشتريها

### 3️⃣ روابط التواصل
- **Discord**: https://discord.gg/Ad66M4CW8v
- **TikTok**: https://www.tiktok.com/@drawstore
- **تسجيل الخروج**: رابط سريع في الفوتر

### 4️⃣ تحسينات اللوقو
- **لا تأثيرات**: تم إزالة التأثيرات Glow و Float
- **تصميم نظيف**: لوقو بسيط وواضح

## 🔧 كيفية الاستخدام

### للمستخدمين:
1. افتح الموقع
2. أدخل مفتاح الترخيص
3. شاهد الملفات المتاحة لك
4. اختر الملف وحمله

### للمطورين - إضافة التوثيق الأساسي:

يجب على Railway API أن يرد بهذا الشكل:

```json
{
  "endpoint": "/verify-license",
  "method": "POST",
  "body": {
    "key": "your-license-key"
  },
  "response": {
    "valid": true,
    "files": ["Bo7", "MW3V1", "OW2"]
  }
}
```

---

## 📝 Railway API Requirements

يجب أن يحتوي Railway API على هذه الـ Endpoints:

### 1. Verify License
```
POST /api/verify-license
Input: { "key": "license-key" }
Output: { "valid": true/false, "files": ["file1", "file2"] }
```

### 2. Get User Files
```
POST /api/get-files
Input: { "key": "license-key" }
Output: { "files": ["file1", "file2"] }
```
